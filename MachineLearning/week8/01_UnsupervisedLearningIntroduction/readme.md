## Clusteringとはなにか
ClusteringはUnsupervisedLearning  
UnsupervisedLearningとは 学習データにラベル(答え, y)が与えられないもの  

Clusteringでは 与えられたデータから データのstructure(構造?)を見付けることで  
データをグルーピングすることができる
